# GitHub Notes

This repository already has an active `v2` branch for the CrowPanel port.

Current remote:

```text
origin https://github.com/liseman/cloudchamber.git
```

## Recommended Workflow

Check status:

```bash
git status
```

Commit changes:

```bash
git add .
git commit -m "Describe the change"
```

Push the v2 branch:

```bash
git push origin v2
```

## Documentation

Main documentation files for the current hardware revision:

- `README.md`
- `INSTALL.md`
- `V2_NOTES.md`

These should describe the CrowPanel-based firmware, not the older Feather-based build.
